

** ThemeData  **

- 정해진 요소들 동일한 스타일 정하는 기술임
- MaterialApp() 안에 theme: ThemeData() 현식으로 저장
- Icon은 => iconTheme, 텍스트는 => textTheme, AppBar는 appBarTheme 사용하면됨


** ThemeData 다른 파일로 옮기기 **

- 그냥 다른 파일 만들기
- import 'package:flutter/material.dart'; 해서
  변수 하나 만들면서 ThemeData 스타일 내용을 작성하기
- main 소스로 './style.dart' import 한다음 그냥 변수를
  스타일로 사용
- 나중에 생길수있는 중복을 방해하기 위해 다음 순서 사용하면됨.
   import './style.dart' as style;
   theme: style.theme,

- 일반 theme 대신 => _theme 사용시 이 변수를 다른파일에서 사용할수없음   

- TextButton() 스타일 필요하면 아래와 같이 사용 (style: TextButton.styleFrom)
  
  textButtonTheme: TextButtonThemeData(
    style: TextButton.styleFrom(

    )
  ),

- 정해진 Container 안에서 다른 스타일하고 싶하면 다른 방법 써야함
  Theme 위쳇 있어야함. data 필수

  body : Theme(
    data : ThemeData(),
    child : Container()
  )



** TAB 만드는 법 **

- 만드는 방식은 3단계 : 
  1번은 state에 tab 현재상태 저장
  2번은 state에 따라 tab 어떻게 보일지 작성
  3번은 사용자가 쉽게 state
